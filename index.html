RedcapAI

Dikelola oleh: RedcapIntermedia

Terakhir diperbarui: [Isi Tanggal, contoh: 1 Juni 2024]

1. Pengantar

Aplikasi RedcapAI dikembangkan oleh RedcapIntermedia untuk memproses data internal hasil penelitian terkait marketing serta data eksternal dari artikel publik. Kebijakan ini menjelaskan bagaimana kami mengelola, menyimpan, dan melindungi data yang diproses.

2. Jenis Data yang Diproses

  • Data Internal: Analisis perilaku konsumen, tren pasar, data historis penjualan.
  • Data Eksternal: Data dari artikel publik seperti studi pasar, laporan industri, dan berita terkait marketing.

Aplikasi tidak mengumpulkan data pribadi kecuali diinput secara eksplisit oleh pengguna untuk keperluan analisis.

3. Tujuan Pengolahan Data

Data digunakan untuk memberikan referensi harga, rekomendasi pasar, dan analisis data yang dimasukkan oleh customer.

4. Pengolahan Data oleh AI

Kami menggunakan model AI pihak ketiga seperti OpenAI, DeepSeek, dan Google Gemini. Data yang diproses dianalisis secara anonim.

5. Keamanan Data

  • Enkripsi data saat transmisi dan penyimpanan menggunakan Google Cloud.
  • Akses terbatas hanya untuk tim yang berwenang.
  • Audit berkala untuk memastikan keamanan data.

6. Berbagi Data

Data hanya dibagikan jika diwajibkan oleh hukum atau untuk integrasi layanan Google Cloud dan AI.

7. Hak Pengguna

  • Meminta akses dan penghapusan data input.
  • Menghubungi kami untuk pertanyaan terkait kebijakan ini.

8. Retensi Data

Data disimpan selama 3 tahun untuk kebutuhan analisis bisnis, sementara data customer dihapus setelah 30 hari.
